# 웹접근성 체크리스트

## 직무별 웹 접근성 점검 항목

> [한국형 웹 콘텐츠 접근성 지침 2.1](https://www.wa.or.kr/board/view.asp?sn=161&page=1&search=&SearchString=&BoardID=0004&cate=)을 기준으로 작성된 직무별 웹 접근성 점검 항목입니다. 이 항목들은 프로젝트 수행 과정에서 웹 접근성 준수 여부를 점검하는 데 도움이 됩니다. 자신의 직무뿐만 아니라 다른 직무의 항목도 함께 확인하여 웹 접근성 준수에 도움이 되도록 합니다.
>
> 추후 개정된 [한국형 웹 콘텐츠 접근성 지침 2.2](https://www.rra.go.kr/ko/reference/kcsList_view.do?nb_seq=5247&nb_type=6)를 기준으로 업데이트될 예정이므로, 해당 문서도 함께 참고하시기를 권장합니다.
